## JSON types 
  * Strings                  | "Hello World"
  * Numbers                  | 10 1.5 -30 1.2e10
  * Booleans                 | true fales
  * null                     | null
  * Arrays                   | [1, 2, 3] ["hello", "world"]
  * Objets                   | {"key": "value", "age": 30}
